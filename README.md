# Quali

Plataforma de capacitación online para pequeñas empresas. Cursos certificados en manipulación de alimentos, EPP, bioseguridad y protocolos de evacuación.

## Sobre el proyecto

Ejercicio práctico de desarrollo web usando **HTML, CSS y JavaScript vanilla** (sin frameworks). Demuestra buenas prácticas de arquitectura frontend, organización de código y estándares de accesibilidad.

### Tecnologías

- HTML5 semántico
- CSS3 (metodología BEM)
- JavaScript vanilla (ES6+)
- Bootstrap 5 (navbar y grid)
- Vite (bundler)

## Instalación

```bash
npm install
```

## Uso

```bash
# Desarrollo
npm run dev

# Build
npm run build

# Preview
npm run preview
```

Servidor en `http://localhost:3000`

## Estructura

```
src/
├── assets/img/       # Imágenes
├── js/modules/       # JavaScript
├── pages/
│   ├── auth/         # Login, registro
│   ├── cursos/       # Cursos
│   └── usuario/      # Panel
├── styles/
│   ├── pages/        # CSS por página
│   └── base.css      # Estilos globales
└── index.html
```
